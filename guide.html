Voice Assistant for Gmail

We have two modes in "Voice assistant for Gmail" application
(A). Command Mode

when you speak any command from the following list then "Voice assistant for Gmail" taking relevant action.

(B). Writing Mode

in this mode, you can write an email using your speech.
how to Start you can start this mode form clicking on mike icon at the bottom of compose an email windows.
How to Stop This mode will autostop when the compose view is closed. then autostart command mode.
